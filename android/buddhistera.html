นโยบายความเป็นส่วนตัว

[ชื่อผู้พัฒนา / บริษัท ] สร้างแอป [ชื่อแอป] เป็นแอป [โอเพนซอร์ซ / ฟรี / ฟรี / การสนับสนุนโฆษณา / เชิงพาณิชย์] SERVICE นี้ให้บริการโดย [ผู้พัฒนา / ชื่อ บริษัท ] [ไม่มีค่าใช้จ่าย] และมีไว้สำหรับการใช้งานตามที่เป็นอยู่

หน้านี้ใช้เพื่อแจ้งผู้เยี่ยมชมเกี่ยวกับนโยบาย [ของฉัน / ของเรา] เกี่ยวกับการเก็บรวบรวมการใช้และการเปิดเผยข้อมูลส่วนบุคคลหากมีใครตัดสินใจที่จะใช้บริการ [ของฉัน / ของเรา]

หากคุณเลือกที่จะใช้บริการ [ของฉัน / ของเรา] แสดงว่าคุณยอมรับการรวบรวมและการใช้ข้อมูลที่เกี่ยวข้องกับนโยบายนี้ ข้อมูลส่วนบุคคลที่ [I / We] รวบรวมใช้สำหรับการให้บริการและปรับปรุงบริการ [I / เรา] จะไม่ใช้หรือแบ่งปันข้อมูลของคุณกับใครยกเว้นตามที่อธิบายไว้ในนโยบายความเป็นส่วนตัวนี้

ข้อกำหนดที่ใช้ในนโยบายความเป็นส่วนตัวนี้มีความหมายเช่นเดียวกับในข้อกำหนดและเงื่อนไขของเราซึ่งสามารถเข้าถึงได้ที่ [ชื่อแอป] เว้นแต่จะระบุไว้เป็นอย่างอื่นในนโยบายความเป็นส่วนตัวนี้

การรวบรวมและการใช้ข้อมูล

เพื่อประสบการณ์ที่ดีกว่าในขณะที่ใช้บริการของเรา [I / เรา] อาจต้องการให้คุณให้ข้อมูลที่สามารถระบุตัวบุคคลได้กับเรา [เพิ่มสิ่งอื่นใดที่คุณรวบรวมได้ที่นี่เช่นชื่อผู้ใช้ที่อยู่สถานที่ตั้งรูปภาพ] ข้อมูลที่ [I / เรา] คำขอจะถูก [เก็บไว้ในอุปกรณ์ของคุณและจะไม่ถูกรวบรวมโดย [ฉัน / เรา] ในทางใดทางหนึ่ง] / [เก็บไว้โดยเราและใช้ตามที่อธิบายไว้ในนโยบายความเป็นส่วนตัวนี้]

แอพใช้บริการของบุคคลที่สามที่อาจรวบรวมข้อมูลที่ใช้ระบุตัวคุณ

ลิงก์ไปที่นโยบายความเป็นส่วนตัวของผู้ให้บริการบุคคลที่สามที่แอพใช้

  • บริการ Google Play
  • AdMob

บันทึกข้อมูล

[I / เรา] ต้องการแจ้งให้คุณทราบว่าเมื่อใดก็ตามที่คุณใช้บริการ [ของฉัน / ของเรา] ในกรณีที่เกิดข้อผิดพลาดในแอป [I / เรา] รวบรวมข้อมูลและข้อมูล (ผ่านผลิตภัณฑ์ของบุคคลที่สาม) บนโทรศัพท์ของคุณ . ข้อมูลบันทึกนี้อาจรวมถึงข้อมูลเช่นที่อยู่อินเทอร์เน็ตโพรโทคอลของอุปกรณ์ (“ IP”), ชื่ออุปกรณ์, รุ่นของระบบปฏิบัติการ, การกำหนดค่าของแอปเมื่อใช้บริการของฉัน / เราเวลาและวันที่ที่คุณใช้บริการ และสถิติอื่น ๆ

คุ้กกี้

คุกกี้คือไฟล์ที่มีข้อมูลจำนวนเล็กน้อยซึ่งโดยทั่วไปจะใช้เป็นตัวระบุที่ไม่ระบุตัวตน สิ่งเหล่านี้จะถูกส่งไปยังเบราว์เซอร์ของคุณจากเว็บไซต์ที่คุณเยี่ยมชมและถูกเก็บไว้ในหน่วยความจำภายในของอุปกรณ์ของคุณ

บริการนี้ไม่ได้ใช้“ คุกกี้” เหล่านี้อย่างชัดเจน อย่างไรก็ตามแอพอาจใช้รหัสบุคคลที่สามและห้องสมุดที่ใช้ "คุกกี้" เพื่อรวบรวมข้อมูลและปรับปรุงบริการของพวกเขา คุณมีตัวเลือกในการยอมรับหรือปฏิเสธคุกกี้เหล่านี้และรู้ว่าเมื่อใดที่มีการส่งคุกกี้ไปยังอุปกรณ์ของคุณ หากคุณเลือกที่จะปฏิเสธคุกกี้ของเราคุณอาจไม่สามารถใช้บางส่วนของบริการนี้

ผู้ให้บริการ

[I / We] อาจจ้าง บริษัท บุคคลภายนอกและบุคคลเนื่องจากเหตุผลดังต่อไปนี้:

  • เพื่ออำนวยความสะดวกบริการของเรา;
  • เพื่อให้บริการในนามของเรา
  • เพื่อดำเนินการบริการที่เกี่ยวข้องกับบริการ; หรือ
  • เพื่อช่วยเราวิเคราะห์การใช้บริการของเรา

[ฉัน / เรา] ต้องการแจ้งให้ผู้ใช้บริการทราบว่าบุคคลที่สามเหล่านี้สามารถเข้าถึงข้อมูลส่วนบุคคลของคุณได้ เหตุผลคือการทำงานที่ได้รับมอบหมายในนามของเรา อย่างไรก็ตามพวกเขามีหน้าที่ต้องไม่เปิดเผยหรือใช้ข้อมูลเพื่อจุดประสงค์อื่นใด

ความปลอดภัย

[I / เรา] ให้ความสำคัญกับความไว้วางใจของคุณในการให้ข้อมูลส่วนบุคคลของคุณแก่เราดังนั้นเราจึงมุ่งมั่นที่จะใช้วิธีการที่เป็นที่ยอมรับในเชิงพาณิชย์ในการปกป้องข้อมูล แต่โปรดจำไว้ว่าไม่มีวิธีการส่งผ่านอินเทอร์เน็ตหรือวิธีการเก็บข้อมูลอิเล็กทรอนิกส์ปลอดภัยและเชื่อถือได้ 100% และ [I / We] ไม่สามารถรับประกันความปลอดภัยแน่นอน

ลิงค์ไปยังเว็บไซต์อื่น

บริการนี้อาจมีลิงค์ไปยังเว็บไซต์อื่น หากคุณคลิกที่ลิงค์บุคคลที่สามคุณจะถูกนำไปยังเว็บไซต์นั้น โปรดทราบว่าเว็บไซต์ภายนอกเหล่านี้ไม่ได้ดำเนินการโดย [me / us] ดังนั้น [I / We] ขอแนะนำให้คุณอ่านนโยบายความเป็นส่วนตัวของเว็บไซต์เหล่านี้ [I / เรา] ไม่มีอำนาจควบคุมและไม่รับผิดชอบต่อเนื้อหานโยบายความเป็นส่วนตัวหรือแนวทางปฏิบัติของเว็บไซต์หรือบริการของบุคคลที่สามใด ๆ

ความเป็นส่วนตัวของเด็ก

บริการเหล่านี้ไม่ได้กล่าวถึงใครที่มีอายุต่ำกว่า 13 ปี [I / เรา] จะไม่รวบรวมข้อมูลส่วนบุคคลที่สามารถระบุตัวบุคคลได้จากเด็กอายุต่ำกว่า 13 ปีในกรณีที่ [I / เรา] ค้นพบว่าเด็กอายุต่ำกว่า 13 ปีจัดหา [me / us] ด้วยข้อมูลส่วนบุคคล [I / We] ลบข้อมูลนี้ออกจากเซิร์ฟเวอร์ของเราทันที หากคุณเป็นพ่อแม่หรือผู้ปกครองและคุณทราบว่าลูกของคุณได้ให้ข้อมูลส่วนบุคคลกับเราโปรดติดต่อ [ฉัน / เรา] เพื่อให้ [I / เรา] สามารถดำเนินการที่จำเป็นได้

การเปลี่ยนแปลงนโยบายความเป็นส่วนตัวนี้

[I / เรา] อาจอัปเดตนโยบายความเป็นส่วนตัวของเราเป็นครั้งคราว ดังนั้นคุณควรตรวจสอบหน้านี้เป็นระยะสำหรับการเปลี่ยนแปลงใด ๆ [ฉัน / เรา] จะแจ้งให้คุณทราบถึงการเปลี่ยนแปลงใด ๆ โดยโพสต์นโยบายความเป็นส่วนตัวใหม่ในหน้านี้
